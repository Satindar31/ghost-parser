<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```bash
git clone https://github.com/satindar31/ghost-parser
```

<p>2. Move the repo directory</p>

```bash
cd ghost-parser
```

<p>3. Run the app using docker</p>

```bash
docker compose up -d
```
